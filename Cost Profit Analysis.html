Cost-Profit Analysis & Prediction

📌 Project Overview

This project analyzes food order data to determine cost, profit trends, and customer behavior. Additionally, a machine learning model predicts profit based on order details such as delivery fees, discounts, and payment methods.

📂 Dataset

  • File: food_orders_new_delhi.csv

  • Columns:

    • Order Value - Total price of an order

    • Delivery Fee - Fee charged for delivery

    • Discounts and Offers - Discount applied

    • Commission Fee - Commission taken by the platform

    • Payment Processing Fee - Transaction charge

    • Refunds/Chargebacks - Amount refunded to customers

    • Payment Method - Mode of payment

    • Meal - Time of order (Breakfast/Lunch/Dinner)

    • Delivery Day - Day on which the order was placed

🔍 Analysis & Insights

  • 94.6% of customers ordered only once, highlighting the need for loyalty programs.

  • Peak ordering days: Tuesday, Saturday, and Monday.

  • Delivery fees are highest for Midnight Meals.

  • Most popular payment method: Cash on Delivery.

  • Profits are impacted by high discounts and commission fees.

📊 Machine Learning Model

  • Algorithm Used: Linear Regression

  • Features Used:

    • Order Value, Delivery Fee, Discounts, Commission Fee, Processing Fee

    • Payment Method, Meal, Delivery Day

  • Performance Metrics:

    • R² Score: Measures model accuracy

    • Mean Squared Error (MSE): Measures error in predictions
